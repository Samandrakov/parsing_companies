## НЕ СДЕЛАНО

### 1. Филиппины — нет финансовых данных (300 компаний)
**Проблема:** Yahoo Finance не индексирует тикеры PSE (.PS). Поиск по названиям тоже не возвращает результатов.
**Решение:** Вручную сопоставить 20-30 крупнейших PSE компаний с их OTC-тикерами на Yahoo (например SM → SVTMF, BDO → BDOUF, AC → AYALY). Это покроет ~80% капитализации PSE. Либо парсить PSE Edge API (есть endpoint stockData.ax, нужно отправлять POST с cmpy_id).

### 2. Мьянма, Камбоджа, Лаос — нет финансовых данных (32 компании)
**Проблема:** Эти биржи не представлены на Yahoo Finance и не имеют открытых API.
**Решение:** Для 32 компаний можно заполнить данные вручную с сайтов бирж. Для диссертации это некритично — рынки крошечные.

### 3. Сингапур — 247/695 без капитализации
**Проблема:** У 247 мелких SGX-компаний не удалось найти тикер Yahoo Finance по названию.
**Решение:** Можно попробовать поиск по ISIN-коду через альтернативные API (OpenFIGI, или SGX Screener). Некритично — покрыты все крупные компании, 448/695 это 64%.

### 4. ISIC-классификация неполная
- Индонезия: 2 компании без ISIC
- Сингапур: 25 без ISIC
- Таиланд: 61 без ISIC
- Малайзия: 3 без ISIC
- Вьетнам: 6 без ISIC
**Решение:** Для оставшихся ~97 компаний можно дополнить классификацию вручную или расширить словарь ключевых слов в `isic_mapper.py`.

### 5. Валюты не приведены к единому знаменателю
**Проблема:** Market Cap хранится в локальных валютах (IDR, THB, SGD, MYR, VND). В consolidated.csv нет колонки в USD.
**Решение:** Добавить конвертацию в USD в `consolidate.py` с фиксированными курсами на дату сбора. В визуализациях (chart 07, 08) конвертация уже есть, но захардкожена в коде.

### 6. Philippines/enrich_pse.py — скрипт написан, но не работает
**Проблема:** Yahoo Finance search API не находит филиппинские компании по имени. Скрипт отрабатывает с 0 результатов.
**Решение:** Переписать на парсинг PSE Edge (POST-запросы на edge.pse.com.ph) или ручное сопоставление тикеров.

## РЕКОМЕНДАЦИИ ДЛЯ ДИССЕРТАЦИИ

1. В методологии указать: источники данных, дату сбора, что классификация автоматическая
2. Вьетнам UPCOM (842 компании) — внебиржевой рынок, для корректного сравнения можно фильтровать по Market_Tier = "Main" (только HOSE, 403 компании)
3. Графики готовы для вставки в работу (charts/*.png)
4. Для анализа "живости" компаний использовать Market_Cap + Volume (если оба > 0, компания активна)
